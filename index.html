NAMI-NYC — Public FAQ Assistant (Demo)

This page previews an embeddable chat widget that answers general questions about NAMI-NYC's helpline, classes, support groups, volunteering, employment, and donating — built for anyone visiting the site, not just people already enrolled in a program.

Open the chat bubble in the bottom-right corner to try it. Ask something like "How do I volunteer?", "What is Family-to-Family?", or "How do I contact the helpline?"

Embedding note: this file is self-contained — copy everything from <div id="nami-widget-root"> down to the closing </script> and paste it before </body> on any page. The live AI response only works inside Claude.ai's artifact preview (it calls Anthropic's API through Claude's built-in proxy with no key needed). For the real naminycmetro.org site, the fetch call needs to point at a small backend endpoint that holds your own Anthropic API key — flagged in the code with TODO(host).
